# ============================================================================
# 1. 参数的完整性分析
# ============================================================================

"""
常见的参数模型及其参数数量：

【标准模型 - 9个参数/joint】(当前实现)
这是最基础的刚体模型，每个joint的参数为：
  1. m       - 质量 (kg)
  2. x_com   - 质心X坐标 (m)
  3. y_com   - 质心Y坐标 (m)
  4. z_com   - 质心Z坐标 (m)
  5. I_xx    - 绕x轴转动惯量 (kg·m²)
  6. I_yy    - 绕y轴转动惯量 (kg·m²)
  7. I_zz    - 绕z轴转动惯量 (kg·m²)
  8. f_v     - 粘性摩擦系数 (N·s/rad)
  9. f_s     - 库伦摩擦/静摩擦 (N·m)

缺陷：
  ✗ 不包含电机惯量
  ✗ 不包含传动比和减速机效应
  ✗ 摩擦模型过于简化

【改进模型 - 12个参数/joint】(推荐用于工业机械臂)
增加三个参数来考虑传动系统：
  1-7.  同上
  8.  J_m     - 电机绕动惯量 (kg·m²)
  9.  r       - 减速比 (dimensionless)
  10. f_v_m   - 电机侧粘性摩擦 (N·m·s/rad)
  11. f_s_m   - 电机侧库伦摩擦 (N·m)
  12. b       - 传动间隙或背隙 (rad)

优点：
  ✓ 考虑了传动系统
  ✓ 更接近实际机械臂
  ✓ 可以建模电机和减速机

【完整模型 - 15-20个参数/joint】(高精度需求)
进一步添加：
  - 粘滞阻尼 (viscous damping)
  - Coulomb 摩擦的速度依赖性
  - 关节柔性 (joint flexibility)
  - 温度依赖性
  - 控制延迟

【流行的标准 - 10个参数/joint】(DH参数+动力学)
很多文献采用的标准参数：
  1. m      - 质量
  2-4. r    - 质心位置 (3个分量)
  5-7. I    - 转动惯量对角元 (3个分量)
  8. I_xy   - 转动惯量非对角元（可选）
  9. f_v    - 粘性摩擦
  10. f_s   - 库伦摩擦
  (或加上 I_xz, I_yz)
"""

# ============================================================================
# 2. 理论模型对比
# ============================================================================

"""
【模型1】最基础模型 - 9个参数
τ = M(q)·ddq + C(q,dq)·dq + g(q) + F_friction

其中摩擦模型：
  F_friction = f_v·dq + f_s·sign(dq)

代码形式：
  θ_i = [m_i, x_i, y_i, z_i, I_xx_i, I_yy_i, I_zz_i, f_v_i, f_s_i]
  参数数量 = 6 joints × 9 = 54个参数


【模型2】包含电机惯量 - 12个参数 (推荐)
τ = M(q)·ddq + C(q,dq)·dq + g(q) + F_friction + τ_motor

关键：必须考虑电机惯量和减速比的影响

从电机侧看：
  τ_motor = J_m·α_motor + f_v_m·ω_motor + f_s_m·sign(ω_motor)

从负载侧看（通过减速比r）：
  ω_motor = r·dq       (电机速度 = 减速比 × 关节速度)
  α_motor = r·ddq      (电机加速度)
  τ_load = τ / r       (力矩传递)

因此：
  τ = M(q)·ddq + C(q,dq)·dq + g(q) 
      + (f_v + f_v_m·r²)·dq + (f_s + f_s_m·r²)·sign(dq)
      + J_m·r²·ddq

重新整理为线性形式：
  τ = [1, ..., 0]·m
      + [q_component]·x_com
      + [q_component]·y_com
      + [q_component]·z_com
      + [q_component]·I_xx
      + [q_component]·I_yy
      + [q_component]·I_zz
      + [ddq]·(J_m·r²)         ← 电机惯量项
      + [dq]·(f_v + f_v_m·r²)  ← 总粘性摩擦
      + [sign(dq)]·(f_s + f_s_m·r²) ← 总库伦摩擦

代码形式：
  θ_i = [m_i, x_i, y_i, z_i, I_xx_i, I_yy_i, I_zz_i, J_eff_i, f_v_i, f_s_i, ...]
  其中：
    J_eff_i = J_m_i · r_i²  (等效电机惯量)
  
  参数数量 = 6 joints × 12 = 72个参数


【模型3】考虑非对角惯量矩阵 - 15个参数
如果不假设质心在关节轴线上，需要考虑：
  I_xx, I_yy, I_zz (3个对角元)
  I_xy, I_xz, I_yz (3个非对角元)

这会增加3个参数，总共12个。

代码形式：
  θ_i = [m_i, x_i, y_i, z_i, I_xx_i, I_yy_i, I_zz_i, I_xy_i, I_xz_i, I_yz_i, 
         J_eff_i, f_v_i, f_s_i, ...]
  参数数量 = 6 joints × (7 + 3 + 3) = 78个参数


【模型4】包含多个传动链 - 可变参数
某些机械臂有多个电机驱动同一个关节，或者有复杂的传动链：
  τ = Σ(J_m_k · r_k² + f_v_m_k · r_k²) · ddq + ...
  
  其中 k = 1, 2, ... n_motors_in_joint
"""

"""
【UR5 (Universal Robots)】
文献中报告的参数数：
  - 基础模型：6 joints × 9 params = 54个参数
  - 加上电机惯量：6 joints × 11 params = 66个参数
  - 包含非对角惯量：6 joints × 15 params = 90个参数

为什么UR没有用完整的15个？因为：
  1. 很多参数不可辨识 (unidentifiable)
  2. 数据噪声导致过拟合
  3. 实际上只有9-11个参数能被准确辨识

【KUKA KR5-R850】
工业应用中通常用：
  - 9个参数/joint （早期模型）
  - 11-12个参数/joint （包含电机和减速机）

【ABB IRB140】
实际辨识通常用：
  - 10参数/joint （基础 + 一个额外参数）
  - 12-13参数/joint （工业版本）

【研究标准】(论文中常见)
  - 最简化: 6个参数/joint (只有质量和惯量)
  - 标准: 9-10个参数/joint
  - 详细: 12-15个参数/joint
  - 完整: 15-20个参数/joint
"""

# ============================================================================
# 4. 参数辨识的实际限制
# ============================================================================

"""
【为什么不能无限增加参数数量？】

1. 信息量限制 (Information Theory)
   ---
   回归矩阵 Φ 的秩受以下限制：
   - rank(Φ) ≤ min(N_samples, N_parameters)
   - 如果 rank(Φ) < N_parameters，系统欠定
   
   例如：
   - 2000个样本，54个参数 → 可以解
   - 2000个样本，200个参数 → 很难准确解

2. 可辨识性 (Identifiability)
   ---
   某些参数组合会导致相同的动力学输出。例如：
   - m 和 J 的某个组合无法分离
   - 电机摩擦和关节摩擦有耦合
   
   结果：
   - 即使有足够的数据，某些参数仍无法准确辨识
   - 通常只有 40-60% 的参数可识别

3. 数据噪声 (Noise)
   ---
   噪声放大：
   - 参数越多，噪声对结果的影响越大
   - Φ 的条件数增大，求解数值不稳定
   - 可能导致参数估计完全错误
   
   条件数：
   - κ = σ_max / σ_min (奇异值比)
   - κ < 100: 良好
   - κ > 1000: 数值不稳定
   - 加更多参数会显著增加κ

4. 采样频率限制 (Sampling Rate)
   ---
   要准确识别电机惯量和高频摩擦，需要：
   - 采样频率 ≥ 1000 Hz（通常只有100-200Hz）
   - 高速运动数据（通常只有中等速度）
"""
